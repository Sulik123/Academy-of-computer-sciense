HTML и CSS для начинающих

Полное руководство по созданию веб-страниц и сайтов. Этот курс включает в себя все основы HTML и CSS, от базового синтаксиса до более сложных тем, таких как адаптивный дизайн.

  • Чему научитесь: создание сайтов с нуля, верстка, работа с блоками и элементами.
  • Длительность: 4 недели.
  • Стоимость: 4000 cом.
  • Преподаватель: Иван Иванов.
Продвинутый JavaScript

Изучите глубины JavaScript для создания сложных веб-приложений. Курс охватывает такие темы, как замыкания, прототипы, асинхронное программирование и многое другое.

  • Чему научитесь: работа с DOM, асинхронный код, оптимизация приложений.
  • Длительность: 5 недель.
  • Стоимость: 5000 сом.
  • Преподаватель: Мария Петрова.
React для начинающих

Создавайте динамичные веб-приложения с помощью React. Курс включает основы JSX, компоненты, управление состоянием и работу с API.

  • Чему научитесь: создание одностраничных приложений (SPA), компоненты, хуки.
  • Длительность: 6 недель.
  • Стоимость: 6000 сом.
  • Преподаватель: Александр Сидоров.
Node.js для начинающих

Обучение созданию серверных приложений на Node.js. Курс охватывает такие темы, как работа с файлами, маршрутизация, подключение к базам данных и многое другое.

  • Чему научитесь: создание серверов, работа с Express, работа с базами данных.
  • Длительность: 4 недели.
  • Стоимость: 6000 сом.
  • Преподаватель: Михаил Романов.
Основы Python

Изучите Python для разработки серверных приложений и машинного обучения. Курс включает основы синтаксиса, работы с данными и основ объектно-ориентированного программирования.

  • Чему научитесь: базовый синтаксис Python, работа с библиотеками, основы ООП.
  • Длительность: 4 недели.
  • Стоимость: 5000 сом.
  • Преподаватель: Ольга Смирнова.
Java для начинающих

Основы программирования на языке Java для создания масштабируемых приложений. Курс включает введение в ООП, основы работы с потоками и коллекциями.

  • Чему научитесь: базовый синтаксис Java, работа с ООП, основы многопоточности.
  • Длительность: 6 недель.
  • Стоимость: 5000 сом.
  • Преподаватель: Елена Васильева.
